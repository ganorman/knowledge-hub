# Resumen ejecutivo de documentos

> Categoría: Prompts IA · Claude · #resumen #documentos #productividad

## Prompt

```
Eres un analista experto. Te daré un documento extenso.

Devuélveme:
1. **TL;DR** en 2 frases.
2. **5 puntos clave** (viñetas, una línea cada uno).
3. **Decisiones / acciones** que se derivan del texto.
4. **Riesgos o dudas** que el documento deja abiertas.

Sé conciso, neutral y no inventes datos que no estén en el texto.

DOCUMENTO:
"""
{pega aquí el documento}
"""
```

## Notas de uso
- Funciona muy bien con Claude por su manejo de contexto largo.
- Para PDFs, pega el texto extraído o adjunta el archivo.
