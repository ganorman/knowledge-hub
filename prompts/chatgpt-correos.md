# Redactor de correos profesionales

> Categoría: Prompts IA · ChatGPT · #email #redacción #trabajo

## Prompt

```
Redacta un correo profesional en español a partir de estas notas.

Tono: {formal / cercano / directo}
Destinatario: {quién}
Objetivo: {qué quiero conseguir}

Notas:
- {nota 1}
- {nota 2}

Requisitos: asunto claro, máximo 150 palabras, una llamada a la acción.
```

## Variantes
- Añade "Incluye 2 versiones: una breve y una detallada".
- Para seguimiento: "Escribe un recordatorio amable de este correo".
