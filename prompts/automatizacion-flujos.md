# Diseñar flujos de automatización

> Categoría: Prompts IA · Automatización · #n8n #make #workflow

## Prompt

```
Quiero automatizar: {objetivo}.
Herramientas disponibles: {n8n / Make / Zapier / Apps Script}.

Diseña el flujo:
1. Disparador (trigger).
2. Pasos intermedios (con la app y la acción en cada uno).
3. Resultado final.
4. Casos límite a vigilar.

Represéntalo como una lista numerada y un diagrama en texto (A → B → C).
```
